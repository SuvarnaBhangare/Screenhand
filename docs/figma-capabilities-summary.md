# Figma Automation — What We Can Build

Based on 131 tested capabilities via Plugin API + CDP browser automation.

---

## Design Creation (via Plugin API)
- **Full website designs** — single scrollable frame (1440x7200px), scrolls like real site in prototype
- **Multi-screen app designs** — mobile (390x844), tablet, desktop frames with navigation
- **Component systems** — buttons, cards, inputs with variants (default/hover/active/disabled)
- **Auto Layout** — responsive flexbox stacks, grids, wrap layouts, fill/hug sizing
- **Real images** — Unsplash photos, avatars, hero images via URL import
- **SVG icons** — Lucide, Heroicons, custom SVGs with recoloring
- **Typography** — Inter family (6 weights), mixed styles in one text block, text styles
- **Color systems** — paint styles, variables, gradients (linear/radial/angular/diamond)

## Visual Effects
- **Glassmorphism** — real backdrop blur + semi-transparent fills (light/dark/tinted)
- **Glow effects** — colored drop shadows, ambient blur orbs
- **Blend modes** — 19 modes (screen, multiply, overlay, etc.)
- **Masks** — star, hex, squircle shaped image crops
- **Boolean ops** — union, subtract, intersect, exclude for custom shapes

## Interactive Prototyping
- **12 trigger types** — click, hover, press, drag, keyboard, timer, mouse enter/leave, media events
- **Smart Animate** — smooth transitions between matched layers (parallax, slides, morphs)
- **Overlays** — modals, tooltips, drawers that appear on top
- **Scroll behavior** — vertical, horizontal carousels, 2D panning, sticky headers
- **Interactive components** — toggles, tab switches, dropdowns, carousels
- **Navigation** — multi-screen flows, swap, back action, URL links, scroll-to anchors
- **12 easings** — linear, ease in/out, bouncy, spring, gentle, quick, slow

## Design System
- **Components & variants** — reusable with property=value naming
- **Design tokens** — paint styles + text styles for consistency
- **Variables** — COLOR, FLOAT, STRING, BOOLEAN with collection binding
- **Layout grids** — 12-col, row, pixel grids stackable
- **Constraints** — responsive positioning (stretch, center, scale)

## Export & Delivery
- **PNG/JPG/SVG/PDF** export at any resolution (1x, 2x, 3x)
- **Prototype links** — shareable interactive demos
- **Dev handoff** — inspectable spacing, colors, typography

## Plugins Available
- **Anima** — Figma to React/HTML/CSS code export
- **Vectary** — 3D models in designs
- **LottieFiles** — Lottie animations
- **Jitter** — motion design & animation export
